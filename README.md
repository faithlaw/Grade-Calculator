# 📊 Grade Calculator
A simple C++ console program where a student can enter their subject 
names and scores, and the program calculates their average, grade, 
and whether they pass or fail!

## 💡 Why this project?
As a student, keeping track of grades across multiple subjects can 
be tricky. This program automates the calculation and gives instant 
feedback — making it a great first C++ project!

## ✨ Features
- Enter any number of subjects (up to 10)
- Input subject name and score for each
- Calculates total and average score
- Assigns a letter grade automatically
- Displays PASS or FAIL status
- Clean summary table at the end

## 🎓 Grading Scale
| Average Score | Grade |
|---------------|-------|
| 90 and above  |  A+   |
| 80 – 89       |   A   |
| 70 – 79       |   B   |
| 60 – 69       |   C   |
| Below 60      |   F   |
